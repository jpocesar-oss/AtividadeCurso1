VERDE VIVO - ONG

Descrição Geral:
A Verde Vivo é um site institucional criado em HTML e CSS para representar uma ONG fictícia dedicada à preservação ambiental e à promoção da sustentabilidade.
O projeto tem como objetivo apresentar informações sobre a organização, seus projetos sociais e permitir o cadastro de novos voluntários.

ESTRUTURA DO PROJETO:

O site é composto por três páginas principais:

index.html – Página inicial com informações sobre a ONG, missão, visão e áreas de atuação.

projetos.html – Página que apresenta os principais projetos da Verde Vivo.

cadastro.html – Página com formulário de cadastro para novos voluntários.

Também há uma pasta de estilos e uma pasta de imagens:

/css/style.css → Arquivo de estilo global.

/img/ → Pasta com imagens utilizadas no site.

FUNCIONALIDADES:

Navegação entre páginas através de um menu de links.

Estrutura semântica utilizando header, main, article, section, nav e footer.

Formulário com campos de entrada: nome, e-mail, telefone, data de nascimento, CEP e endereço.

Campo de seleção de gênero.

Layout simples, limpo e acessível.

TECNOLOGIAS UTILIZADAS:

HTML5: estrutura e conteúdo do site.

CSS3: estilização e layout visual.

COMO EXECUTAR O PROJETO:

Baixe ou copie os arquivos do projeto.

Abra a pasta do projeto no seu editor de código (como Visual Studio Code).

Execute o arquivo "index.html" em um navegador para visualizar o site.
